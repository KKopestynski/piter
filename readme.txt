Urodziny Pitera - README

🎉 Opis Projektu

Strona "Urodziny Pitera" to kolorowa, dynamiczna i interaktywna strona internetowa stworzona na cześć urodzin Piotra. Zawiera animacje, efekty wizualne i specjalny prezent w formie osadzonego pliku PDF.

🚀 Funkcjonalności

Tło z teksturą - Strona posiada wzorzyste tło dodające efekt wizualny.

Marquee - Przesuwający się tekst z życzeniami urodzinowymi.

Efekty animacyjne:

Migający tekst

Rotujący tekst

Poruszające się obrazy (poziomo, pionowo, po przekątnej, odbijające się)

Osadzony PDF - Niespodzianka ukryta w prezencie.

Responsywność - Strona dostosowuje się do urządzeń mobilnych.

📁 Struktura Plików

index.html - Główny plik HTML zawierający całą stronę.

sample.pdf - Plik PDF z niespodzianką dla Pitera.

Obrazy GIF z animacjami.

🎨 Technologie

HTML

CSS (animacje i efekty wizualne)

📜 Instrukcja Uruchomienia

Pobierz pliki projektu.

Otwórz index.html w przeglądarce internetowej.

📱 Obsługa na Urządzeniach Mobilnych

Strona jest dostosowana do ekranów mobilnych poprzez media queries.
